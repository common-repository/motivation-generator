== Description ==

Motivation Generator allows you to create (de)motivational posters right on your website

== Installation ==

1. Unpack .zip to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Display plugin via shortcode [motgen_plugin]
